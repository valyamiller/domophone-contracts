# Учёт договоров домофонных трубок

Веб-приложение для учёта договоров на обслуживание домофонных трубок.

## Возможности

- Авторизация с разграничением прав (4 роли: admin, senior, manager, viewer)
- Управление клиентами (добавление, продление договоров)
- Автоматическая генерация номеров договоров и лицевых счетов
- Поиск клиентов по всем полям
- Оповещения об окончании договоров (за 30, 14, 7 дней)
- История оплат
- Генерация и печать договоров в формате DOCX
- Отчёты с экспортом в Excel

## Технологии

- Python 3.11+
- Flask
- SQLite
- Flask-Login
- python-docx
- openpyxl
